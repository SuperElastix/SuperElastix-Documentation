Run "make html renderuml" to build the docs with with creation of the uml diagrams.
This requires plantuml.jar to be in this directory, which can be downloaded from http://plantuml.com/download.

UML rendering tested for:

java -jar plantuml.jar -version
PlantUML version 1.2017.19 (Sun Nov 12 12:46:22 CET 2017)
(GPL source distribution)
Dot version: dot - graphviz version 2.38.0 (20140413.2041)
